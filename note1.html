Two methods to implement animation effect

study notes     2018-07-05
First method is adding the animation attribute in css3. The code likes that:
.modalpic{
 -webkit-animation: first 0.5s;
 -o-animation: first 0.5s;
 animation: first 0.5s;
}
@keyframes first{
 from{transform: scale(0.5);}
 to{transform: scale(1);}
}
 Animation attribute has 6 attributes:
  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
Another method is to use ScrollReveal.js.
1、introduce the js file
<script src="js/scrollReveal.js"></script>
 2、change in HTML file
The elements which want animation must add data-sroll-reveal attribute, like:
<div data-scroll-reveal="enter left and move 50px over 1.33s">blog</div>
<div data-scroll-reveal="enter from the bottom after 1s">ScrollReveal</div>>
<div data-scroll-reveal="wait 2.5s and then ease-in-out 100px">ScrollReveal</div>
 3、JavaScript
<script>
 if (!(/msie [6|7|8|9]/i.test(navigator.userAgent))){
  (function(){
   window.scrollReveal = new scrollReveal({reset: true});
  })();
 };
</script>
 4、data-scroll-reveal attribute
We can define data-scroll-reveal attribute ourselves. These are the keywords and values of the attributes.
  • enter:
    Shows the initial direction
    value: top | right | bottom | left
  • move:
    Shows the distance the element need to move
    value: number(px)
  • over:
    Duration of the animation
    value: number(s)
  • after/wait:
    The delay of the animation
    value: number(s)
We can also add some statements in the data-scroll-reveal attribute and scrollReveal.js supports these statements:
  • from
  • the
  • and
  • then
  • but
  • with
