How to show my own pages in the GitHub?

study notes     2018-07-06
if we don’t buy our own domain, we can use GitHub to show the pages.
My computer is win10 and i have installed bash in my laptop.
  • First, create a new repository with nothing in the repository in your GitHub.
  • Then, run the terminal and bash.
  • Find the dictionary where you want to clone the repository and run the git clone url.
  • Copy the html, css, and other files in the dictionary.
  • Run the command in the terminal.
    Git add --all
    Git commit -m “melody”
    Git push origin master
  • And then check if there are files in the GitHub in the website.
  • Find setting tag and click.
  • Scroll down and find GitHub Pages.
  • Choose the none to master branch and then click save.
  • Then your website is done.
